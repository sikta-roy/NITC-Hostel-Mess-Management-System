## Image Guidelines

- **Format**: PNG or JPG
- **Recommended Size**: 1920x1080 or similar aspect ratio
- **File Size**: Keep under 1MB per image for better loading
- **Naming**: Use lowercase with hyphens (e.g., `student-dashboard.png`)
